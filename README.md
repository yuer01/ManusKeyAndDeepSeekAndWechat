# 愿景和目标
助力自由职业和上班族巧用 AI 赋能工作并拓展自媒体业务


# ⭐️  活动内容形式：
1. 轮流自我介绍，相互熟悉下，中间别人可以提问（20分钟）。
2. 组织者或参与人会现场讲一些操作教程，比如：DeepSeek本地部署，影刀RPA的使用等。（20分钟）
3. 自由讨论，这个环节时间最长，参加都是在这个方面兴趣点比较丰富。一般会持续到活动结束。（一个半小时）
4. 最后一小时自由安排，有事儿的可以先回，想多了解的多聊会。
5. 线下活动结束，群里会发讨论大纲（去除个人隐私部分）。
